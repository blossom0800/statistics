# 통계학은 주방장의 칼과 같음. 한식 조리사에겐 한식을 만드는 칼이 되고 일식 조리사에겐 일식을 만드는 칼이 됨

```
 1) 핵심 이슈 정리 (전처리와 같이 조사할 부분을 먼저 잡고 그에 맞는 데이터를 정리)
 2) 조사 대상 선별
 3) 필요 데이터 규명, 실험/조사 방법 수립 (#1~3. 실험계획법)
 4) 데이터 수집, 정리, 통계 처리  정보
 5) 중요한 요인(factor) 발견
    A.	머신러닝과 딥러닝의 차이는? : 자동차를 데이터로 집어넣는다면..사람이 자동차의 factor를 분류한다면(바퀴, 브랜드, 연비..) 그건 머신러닝. 딥러닝은 차의 factor들을 기계가 스스로 선택하는 것.
    B.	머신러닝 비슷한 것(사람 기준), 딥러닝은 개념적으로 비슷한 것을 스스로 학습함
    C.	딥러닝 Feature를 기계가 스스로 선택하기 때문에 black box라고도 불림 -> 결과를 예측할 수 없음
    D.	사람에게 맡길지 기계에게 맡길지에 따라 선택함
 6) 통계적 모형 개발, 추정
 7) 적합성 검증, 모형 최적화 -> 해답을 찾는 것
 8) 결론 도출, 의사결정 제안
```
